MY REMOTE COURSES BLOCK PLUGIN
=================================

My remote courses is a Moodle 2.X block that shows remote courses with their overview activities.
You can hide or show each activity overview by clicking on their icon.

Web: https://github.com/IOC/moodle-block-myremotecourses


Install
=======
Download code from gitHub, and copy inside directory MOODLE_PATH/blocks/myremotecourses


Configuration
=============
To connect my remote courses block with a Moodle 1.X you need to provide ajax.php path on
the remote host.

My remote courses block allows to show courses from only one remote host, you need to select
which one of remote hosts available is going to be used.
